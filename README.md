# Daily Affirmation - Web App (PWA)

Start your day with positivity. Get beautiful affirmations and inspiring images, automatically translated for you.

This repository hosts the Progressive Web App build of Daily Affirmation, served at [daffirm.soyames.com](https://daffirm.soyames.com).

- Works in any modern browser, no installation required
- Installable on your home screen (Android, iOS, and desktop)
- Offline support after first load
- Available in English, French, Spanish, German, Arabic, and Chinese

Prefer the native Android app? Get the APK from the [downloads page](https://soyames.com/downloads).
